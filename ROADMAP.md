## 开发节奏

- **每周迭代**: 每周发布一个小版本
- **功能优先**: 先做核心功能，再做辅助功能
- **持续优化**: 根据反馈不断改进

## 当前任务

### Sprint 1: 基础框架 (v0.1.0)
**目标**: 搭建项目基础框架，实现基础 API

**任务清单**:
1. 搭建 FastAPI 项目结构

3. 实现用户认证模块
4. 实现基础 CRUD 接口
5. 编写单元测试

**验收标准**:
- API 可以正常启动

- 基础接口可以正常调用
- 测试覆盖率 > 60%

## 技术决策

### 数据库选择
- **主数据库**: PostgreSQL (结构化数据)
- **缓存**: Redis (热点数据缓存)
- **文件存储**: 本地/云存储 (图片等)

### AI 模型选择
- **图像识别**: MiMo Vision / OpenAI GPT-4V
- **文本分析**: MiMo / Claude
- **知识问答**: RAG + 向量数据库

### 部署方案
- **开发环境**: 本地 Docker
- **测试环境**: 云服务器
- **生产环境**: K8s + Docker
